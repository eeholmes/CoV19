Forecasts

These are forecast by fitting a log-normal shape curve the positive cases and estimating deaths from the 7-lag CFR. Plot are only shown if the confidence intervals on the peak can be computed. The 95% confidence intervals on that peak is the grey line at the top. If it is wide, it means there is no information in the data on the location of the peak, aka we have no clue what will happpen. Reasons for this:

  • The curve has not started to flatten.
  • The data are bad (too noisy). The WA data is an example.
